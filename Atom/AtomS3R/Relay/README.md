# ATOMS3R Relay 入力スイッチ

M5Stack ATOMS3RとM5Stackリレーユニットを使用し、複数の入力方法からリレーを操作できる入力スイッチです。

## 主な機能

- Wi-Fiブラウザから操作
- BLEから操作
- 本体ボタンから操作
- 外部スイッチから操作

リレーの動作方法は以下の3種類から選択できます。

- ダイレクトモード
- ラッチモード
- タイマーモード

## 使用したもの

- M5Stack ATOMS3R
- ATOM PortABC拡張ベース
- M5Stack用ミニリレーユニット [U023]
- M5Stack用エンコーダユニット
- 3.5ｍｍプラグ（外部出力に使用する際）
- モード切替ボタン（以下を使用）  
    Grove - デュポン変換ケーブル  
    タクトスイッチ  
    ブレッドボード
- 外部スイッチ入力を行うときはGPIO8とGをつなげるための部品が別途必要

## 操作モード

### ダイレクトモード

入力している間だけリレーをONにします。

### ラッチモード

1回入力するとリレーがONになります。
次に入力するとOFFになります。

### タイマーモード

1回入力すると、設定した時間だけリレーをONにします。

## タイマー設定

タイマー時間は以下の方法で変更できます。

- エンコーダ
- Wi-Fiブラウザ
- BLE

設定したタイマー時間は本体に保存され、電源を切っても保持されます。

## 画面表示

ATOMS3Rの画面には以下を表示します。

- 現在の動作モード
- Wi-Fi / BLE通信モード
- 通信相手の接続状態（接続中は緑丸で表示）
- タイマー設定時間
- タイマー動作中のカウントダウン

## 通信

Wi-Fiではブラウザからリレー操作やモード変更、タイマー設定ができます。

BLEでは専用クライアントから同様の操作ができます。
