# M5Atom-Projects


> [!IMPORTANT]
> **このプログラムは AtomS3Lite 受信機用です。**
>
> リモコンで使用するには、同じリポジトリ内の
>
> `M5Stick / M5StickS3 / Relay / M5StickS3_Relay_Remote.ino`
>
> をM5StickS3へ書き込んで組み合わせて使用してください。
## 使用したもの

- AtomS3Lite・・・本体
- M5Stack用ミニリレーユニット [U023]・・・付属のHY2.0-4P（Grove互換）ケーブルで接続
- TailBAT・・・AtomS3Liteに挿す電源（またはUSB Type-C給電）
- 3.5mmプラグ・・・リレー出力に使用

## 使い方

### ESP-NOW

#### ペアリング

- AtomS3Lite本体ボタンを約4秒長押しすると、青色に点滅し約15秒間ペアリング待機になります。
- M5StickS3リモコンから新規登録を行います。
- 一度登録すると、以後は自動で接続されます。

#### 操作

- M5StickS3リモコンからリレー操作、モード変更、タイマー時間の設定ができます。

### Wi-Fi

- 初期設定はAtomRelayで始まるSSIDへ接続します。
- パスワードを入力します。（デフォルト：`00000000`）
- ブラウザで `http://192.168.4.1/` を開きます。
- リレー操作、モード変更、タイマー時間の設定、Wi-Fi名・BLE名の変更ができます。


### BLE

- Web Bluetooth対応ブラウザで「（BLE）コンパクト無線スイッチ.html」を開いて接続できます。
- Wi-Fiと同様に、リレー操作、モード変更、タイマー時間の設定、Wi-Fi名・BLE名の変更ができます。


## 操作方法

### ダイレクトモード

- ボタンを押している間だけリレーONになります。LEDは青。

### ラッチモード

- ボタンを1回押すとリレーONになります。LEDは緑。
- もう1回押すとリレーOFFになります。

### タイマーモード

- ボタンを1回押すと、設定時間だけリレーONになります。LEDは黄。

### タイマー時間変更

以下の方法から変更できます。

- M5StickS3リモコン（ESP-NOW）
- Wi-Fi
- BLE

設定した時間は本体に保存され、電源を切っても保持されます。
