# M5Atom-Projects

## 使用したもの

- AtomS3Lite・・・本体
- M5Stack用ミニリレーユニット [U023]・・・AtomS3LiteとGrove接続
- TailBAT・・・AtomS3Liteに挿す電源
- 3.5mmプラグ・・・リレーユニットと接続



## 使い方

### モード切替
- ダイレクトモード（青）・・・ボタンを押している間リレーON（赤）  
  &emsp;設定方法：リセットボタンを押して、離す。

- ラッチモード（緑）・・・ボタンを一度押すとリレーON（赤）、もう一度押すとリレーOFF  
  &emsp;設定方法：本体ボタンを押しながらリセットボタンを押して、離す。リセットボタンを離したら本体ボタンから手を離す。


  以下は未編集
### Wi-Fi

- Wi-Fiをつなぐ。SSIDは「AtomServo」で始まる名前
- 自分で設定したパスワードを入れる（コード内を書き換えます。デフォルトは `00000000`）
- ブラウザを開き、`http://192.168.4.1/` を開く
- ボタンを押すとサーボが1往復する

### BLE

- アプリ：nRF Connect for MobileでiPhoneで動作確認済み
- 現在は、より簡単な操作方法を検討中

### 物理

- AtomLite本体ボタンで動作
- G22ピンに3.5mmジャックを接続する
  - GND（黒線）はブレッドボードなどで分配してGへ接続する
  - 外部スイッチを接続するとサーボが1往復する
    
 
## 操作方法

### 回転角度調整（通常モード）

- 赤LEDがついている状態で回転角ユニットを回して角度を決められる。

### 回転速度（速度調整モード）

- G39（本体ボタン）を長押しで青LEDがついている状態で回転角ユニットを回して往復の速度を決められる。
- 青LED点灯時にG39短押しでサーボ往復し、速度を見ながら調整可能
- G39長押しで通常モードに戻れる
