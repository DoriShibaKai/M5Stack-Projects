```markdown
# M5Atom-Projects

## 使用したもの

- AtomLite・・・本体
- M5Stack用回転角ユニット・・・AtomLiteとGrove接続
- TailBAT・・・AtomLiteに挿す電源
- SG90サーボモータ・・・AtomLiteとピン接続
- 3.5mmジャック・・・AtomLiteのピンと接続（Gはブレッドボードで分配）

## モード切替
- 電源を入れる・リセットボタンを押すで、Wi-Fiモード（青）
- 本体ボタンを押す、またはBUTTON_PIN1で示すピンに入力しながら電源を入れる・リセットボタンを押すで、BLEモード（緑）
- どちらのモードでも物理スイッチは有効

## モードの説明
### Wi-Fi
- Wi-Fiをつなぐ。SSIDは「AtomServo」で始まる名前
  - SSID の末尾は `ESP.getEfuseMac()` の下位 24bit（chipid）を 6桁の16進数にしたものです  
    ※ 完全な MAC アドレスは使用していません
- 自分で設定したパスワードを入れる（コード内を書き換えます。デフォルトは `00000000`）
- ブラウザを開き、`http://192.168.4.1/` を開く
- ボタンを押すとサーボが1往復する


### BLE
- アプリ：nRF Connect for MobileでiPhoneで動作確認済み（474Fを入力）
- 現在は、より簡単な操作方法を検討中

### 物理
- AtomLite本体ボタンで動作
- BUTTON_PIN1で示すピンに3.5mmジャックを接続する
  - GND（黒線）はブレッドボードなどで分配してGへ接続する
  - 外部スイッチを接続するとサーボが1往復する
```
