# M5Atom-Projects

M5Stack ATOM Liteを使い、サーボモータを複数の入力方法から1往復動作させるプロジェクトです。

対応する入力方法：

- Wi-Fiブラウザ
- BLE
- ATOM Lite本体ボタン
- 外部スイッチ

M5Stack用回転角ユニットに加え、Wi-Fiブラウザ画面やBLE操作用HTMLからも回転角度と回転速度を調整できます。


## 使用したもの

- ATOM Lite
- M5Stack用回転角ユニット
- TailBAT
- SG90サーボモータ
- 3.5mmジャック（外部スイッチ接続用）
- 必要に応じてブレッドボード等（GND分配・サーボモータ接続用）


## 操作モード

### 通常モード

サーボの回転角度を調整するモードです。

- 待機LED：緑
- サーボ動作中：赤

通常の電源ON・リセットで起動します。


### 速度調整モード

サーボの回転速度を調整するモードです。

- 待機LED：黄
- サーボ動作中：赤

以下のどちらかを入力しながら電源ON・リセットすると起動します。

- ATOM Lite本体ボタン
- `BUTTON_PIN1` で指定した外部入力

起動時に使用したボタン入力は、サーボ動作入力として扱われません。


## 回転角度・回転速度の調整

回転角度と回転速度は、以下から調整できます。

- M5Stack用回転角ユニット
- Wi-Fiブラウザ画面
- BLE操作用HTML

通常モードでは回転角度、速度調整モードでは回転速度を回転角ユニットで調整します。


## 設定の保存

以下は保存されます。

- 回転角度
- 回転速度
- Wi-Fi / BLE通信モード

通常モード / 速度調整モード自体は保存されず、起動方法によって決まります。


## Wi-Fi / BLE通信モード

Wi-FiとBLEは同時起動しません。

- Wi-Fiモード：BLEを起動しない
- BLEモード：Wi-Fi APとブラウザ操作用Webサーバーを起動しない

Wi-Fi / BLEの選択状態は保存され、次回も前回の通信モードで起動します。


## Wi-Fiモード

### 起動表示

1. 白LEDが2回点滅
2. 通信準備中は消灯
3. 操作可能になると緑または黄を点灯


### 接続方法

1. Wi-Fi一覧から `AtomServo_XXXXXX` に接続
2. コード内で設定したパスワードを入力
3. ブラウザで `http://192.168.4.1/` を開く

SSID末尾の `XXXXXX` は、`ESP.getEfuseMac()` から取得した値の下位24bitを6桁の16進数にしたものです。完全なMACアドレスはSSIDには使用していません。

★ SSIDが出ないときは、本体のリセットボタンを押してください。


### パスワード

コード内で設定します。

```cpp
const char* password = "00000000";
```

Wi-Fiパスワードは8文字以上にしてください。


### Wi-Fiチャンネル

周辺Wi-Fiをスキャンし、`1ch / 6ch / 11ch` から比較的混雑の少ないチャンネルを自動選択します。

最大接続数は2台です。


## BLEモード

### 起動表示

1. 青LEDが2回点滅
2. 通信準備中は消灯
3. 操作可能になると緑または黄を点灯


### BLE接続

BLE名：

```text
AtomServo
```

対応するCharacteristicへ以下の文字列を送信します。

| 送信文字列 | 動作 |
|---|---|
| `GO` | サーボを1往復 |
| `NORMAL` | 通常モードへ切り替え |
| `SPEED` | 速度調整モードへ切り替え |
| `ANGLE:数値` | 回転角度を設定 |
| `SPEEDVALUE:数値` | 回転速度を設定 |
| `WIFI` | Wi-Fiモードへ切り替え |


### BLE UUID

Service UUID：

```text
4FAFC201-1FB5-459E-8FCC-C5C9C331914B
```

Characteristic UUID：

```text
BEB5483E-36E1-4688-B7F5-EA07361B26A8
```

BLE動作確認：

- nRF Connect for Mobile
- Windows Chrome用BLE操作HTML


## BLE操作用HTML

BLE操作用のAtomServo_BLE.HTMLファイルは公開中

対応ブラウザからHTMLファイルを開き、ATOM Liteの `AtomServo` に接続して操作します。

主な操作：

- BLE接続
- サーボを動かす
- 通常モードへ切り替え
- 速度調整モードへ切り替え
- 回転角度調整
- 回転速度調整
- Wi-Fiモードへ切り替え

※ Web Bluetooth対応環境が必要です。動作確認はWindows版Chromeで行っています。


## Wi-Fi / BLEの切り替え

### Wi-Fi → BLE

Wi-Fiブラウザ画面からBLEモードへ切り替えます。


### BLE → Wi-Fi

BLEから `WIFI` を送信します。

BLE操作用HTMLからもWi-Fiモードへ切り替えられます。


### 補助的な切り替え方法

通常の通信操作ができない場合に備え、本体ボタンによる切り替え方法も実装しています。

**ATOM Lite本体ボタンを10秒以上長押しすると、現在とは反対の通信モードへ切り替わります。**

- Wi-Fiモード中 → BLEモードへ切り替え
- BLEモード中 → Wi-Fiモードへ切り替え

切り替え時は通信モードを保存し、LEDで切り替え先を表示した後、ATOM Liteが自動再起動します。

- BLEへ切り替え：青2回点滅
- Wi-Fiへ切り替え：白2回点滅

この操作は、ブラウザやBLEから通信モードを切り替えられない場合の補助的な方法を想定しています。


## 物理入力

以下の入力でサーボを1往復させます。

- ATOM Lite本体ボタン
- `BUTTON_PIN1` で指定した外部入力

例：

```cpp
const int BUTTON_PIN1 = 22;
```

この場合はGPIO22を使用します。`BUTTON_PIN1` は使用環境に応じて変更できます。


### 3.5mmジャック接続例

- 信号線：`BUTTON_PIN1` で指定したGPIO
- GND線：GND

必要に応じてブレッドボード等でGNDを分配します。


## サーボ動作

1回の入力につき、サーボが1往復します。

`millis()` を使用したノンブロッキング制御で、動作中の追加入力は受け付けず、連打分もキューに溜めません。

1往復完了後、新しい入力を受け付けます。


## LED表示

| 状態 | LED |
|---|---|
| Wi-Fi起動 | 白2回点滅 |
| BLE起動 | 青2回点滅 |
| 通信準備中 | 消灯 |
| 通常モード待機 | 緑 |
| 速度調整モード待機 | 黄 |
| サーボ動作中 | 赤 |
| 長押しでBLEへ切り替え | 青2回点滅 |
| 長押しでWi-Fiへ切り替え | 白2回点滅 |

## 無線が反応しないとき

リセットボタンを押して、起動時の点滅色を確認してください。

- 白2回点滅：Wi-Fiモード
- 青2回点滅：BLEモード

Wi-FiとBLEは同時起動せず、点滅色に応じた無線通信のみが有効になります。

通常の通信操作で切り替えられない場合は、ATOM Lite本体ボタンを10秒以上長押しすると、現在とは反対の通信モードへ切り替えられます。
