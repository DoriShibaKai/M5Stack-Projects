# M5Atom-Projects

M5Stack ATOM Liteを使い、サーボモータを複数の入力方法から1往復動作させるプロジェクトです。

対応する入力方法：

- Wi-Fiブラウザ
- BLE
- ATOM Lite本体ボタン
- 外部スイッチ

M5Stack用回転角ユニットで、回転角度と回転速度を調整できます。


## 使用したもの

- ATOM Lite
- M5Stack用回転角ユニット
- TailBAT
- SG90サーボモータ
- 3.5mmジャック（外部スイッチ接続用）
- 必要に応じてブレッドボード等（GND分配・サーボモータ接続用）


## 操作モード

### 通常モード

回転角ユニットでサーボの回転角度を調整します。

- 待機LED：緑
- サーボ動作中：赤

通常の電源ON・リセットで起動します。


### 速度調整モード

回転角ユニットでサーボの回転速度を調整します。

- 待機LED：黄
- サーボ動作中：赤

以下のどちらかを入力しながら電源ON・リセットすると起動します。

- ATOM Lite本体ボタン
- `BUTTON_PIN1` で指定した外部入力

起動時に使用したボタン入力は、サーボ動作入力として扱われません。


## 設定の保存

以下は保存されます。

- 回転角度
- 回転速度
- Wi-Fi / BLE通信モード

通常モード / 速度調整モード自体は保存されず、起動方法によって決まります。


## Wi-Fi / BLE通信モード

Wi-FiとBLEは同時起動しません。

- Wi-Fiモード：BLEを起動しない
- BLEモード：Wi-Fi APとブラウザ操作用Webサーバーを起動しない

Wi-Fi / BLEの選択状態は保存され、次回も前回の通信モードで起動します。


## Wi-Fiモード

### 起動表示

1. 白LEDが2回点滅
2. 通信準備中は消灯
3. 操作可能になると緑または黄を点灯


### 接続方法

1. Wi-Fi一覧から `AtomServo_XXXXXX` に接続
2. コード内で設定したパスワードを入力
3. ブラウザで `http://192.168.4.1/` を開く

SSID末尾の `XXXXXX` は、`ESP.getEfuseMac()` から取得した値の下位24bitを6桁の16進数にしたものです。完全なMACアドレスはSSIDには使用していません。

★ SSIDが出ないときは、本体のリセットボタンを押してください。


### パスワード

コード内で設定します。

```cpp
const char* password = "00000000";
```

Wi-Fiパスワードは8文字以上にしてください。


### ブラウザ操作

- 動かす
- 通常モードへ切り替え
- 速度調整モードへ切り替え
- BLEモードへ切り替え


### Wi-Fiチャンネル

周辺Wi-Fiをスキャンし、`1ch / 6ch / 11ch` から比較的混雑の少ないチャンネルを自動選択します。

最大接続数は2台です。


## BLEモード

### 起動表示

1. 青LEDが2回点滅
2. 通信準備中は消灯
3. 操作可能になると緑または黄を点灯


### BLE接続

BLE名：

```text
AtomServo
```

対応するCharacteristicへ以下の文字列を送信します。

| 送信文字列 | 動作 |
|---|---|
| `GO` | サーボを1往復 |
| `NORMAL` | 通常モードへ切り替え |
| `SPEED` | 速度調整モードへ切り替え |
| `WIFI` | Wi-Fiモードへ切り替え |


### BLE UUID

Service UUID：

```text
4FAFC201-1FB5-459E-8FCC-C5C9C331914B
```

Characteristic UUID：

```text
BEB5483E-36E1-4688-B7F5-EA07361B26A8
```

- BLE動作確認：nRF Connect for Mobile、Windows Chrome用BLE操作HTML


### Windows Chrome

BLE操作用HTMLから以下を操作できます。

- BLE接続
- 動かす
- 通常モードへ切り替え
- 速度調整モードへ切り替え
- Wi-Fiモードへ戻す


## Wi-Fi / BLEの切り替え

### Wi-Fi → BLE

Wi-Fiブラウザ画面の「BLEモードへ切替」を押します。

### BLE → Wi-Fi

BLEから `WIFI` を送信します。BLE操作用HTMLでは「Wi-Fiモードへ戻す」を押します。

どちらも通信モード保存後、ATOM Liteが自動再起動します。


## 物理入力

以下の入力でサーボを1往復させます。

- ATOM Lite本体ボタン
- `BUTTON_PIN1` で指定した外部入力

例：

```cpp
const int BUTTON_PIN1 = 22;
```

この場合はGPIO22を使用します。`BUTTON_PIN1` は使用環境に応じて変更できます。


### 3.5mmジャック接続例

- 信号線：`BUTTON_PIN1` で指定したGPIO
- GND線：GND

必要に応じてブレッドボード等でGNDを分配します。


## サーボ動作

1回の入力につき、サーボが1往復します。

`millis()` を使用したノンブロッキング制御で、動作中の追加入力は受け付けず、連打分もキューに溜めません。

1往復完了後、新しい入力を受け付けます。


## LED表示

| 状態 | LED |
|---|---|
| Wi-Fi起動 | 白2回点滅 |
| BLE起動 | 青2回点滅 |
| 通信準備中 | 消灯 |
| 通常モード待機 | 緑 |
| 速度調整モード待機 | 黄 |
| サーボ動作中 | 赤 |

## 無線が反応しないとき

リセットボタンを押して、起動時の点滅色を確認してください。

- 白2回点滅：Wi-Fiモード
- 青2回点滅：BLEモード

Wi-FiとBLEは同時起動せず、点滅色に応じた無線通信のみが有効になります。
