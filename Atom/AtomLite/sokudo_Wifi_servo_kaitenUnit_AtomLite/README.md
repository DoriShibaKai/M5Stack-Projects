# M5Atom-Projects

M5Stack ATOM Liteを使って、サーボモータを複数の入力方法から1往復動作させるプロジェクトです。

以下の入力に対応しています。

- Wi-Fiブラウザ操作
- BLE操作
- ATOM Lite本体ボタン
- 外部スイッチ入力

回転角度と回転速度は、M5Stack用回転角ユニットで調整できます。


## 使用したもの

- ATOM Lite・・・本体
- M5Stack用回転角ユニット・・・ATOM LiteとGrove接続
- TailBAT・・・ATOM Liteに挿す電源
- SG90サーボモータ・・・ATOM Liteとピン接続
- 3.5mmジャック・・・外部スイッチ接続用
- 必要に応じてブレッドボード等・・・GND分配・サーボモータ接続用


## 主な機能

- サーボモータの1往復動作
- 回転角度調整
- 回転速度調整
- Wi-Fiブラウザ操作
- BLE操作
- 本体ボタン操作
- 外部スイッチ操作
- Wi-Fi / BLEの切り替え
- Wi-Fi / BLEモードの保存
- 回転角度設定の保存
- 回転速度設定の保存
- サーボ動作中の追加入力を無視
- Wi-FiとBLEの同時起動を避ける排他動作


## 動作モード

本機には、以下の2種類の「操作モード」があります。

### 通常モード

回転角ユニットで、サーボの回転角度を調整します。

- 待機LED：緑
- 回転角ユニット：回転角度調整
- サーボ動作中：赤

### 速度調整モード

回転角ユニットで、サーボの回転速度を調整します。

- 待機LED：黄
- 回転角ユニット：回転速度調整
- サーボ動作中：赤


## 通常モード / 速度調整モードの選択

### 通常モードで起動

そのまま、

- 電源を入れる
- リセットする

ことで通常モードになります。

待機LEDは緑です。

### 速度調整モードで起動

以下のどちらかを入力した状態で、

- ATOM Lite本体ボタン
- `BUTTON_PIN1` で指定した外部入力

電源を入れる、またはリセットします。

速度調整モードで起動し、待機LEDが黄になります。

起動時に使用したボタン入力は、サーボ動作入力として扱われません。


## 設定値の保存

以下の設定値は保存されます。

- 回転角度
- 回転速度
- Wi-Fi / BLE通信モード

そのため、通常モードで設定した回転角度は、速度調整モードでも維持されます。

また、速度調整モードで設定した回転速度は、通常モードでも維持されます。


## Wi-Fi / BLE通信モード

Wi-FiとBLEは同時起動しません。

- Wi-FiモードではBLEを起動しない
- BLEモードではWi-Fi APと、ブラウザ操作画面を提供するWebサーバーを起動しない

これにより、通信の競合や不安定化を避けています。

Wi-Fi / BLEの選択状態は保存されるため、次回起動時も前回の通信モードで起動します。


## Wi-Fiモード

### 起動表示

Wi-Fiモード起動時は、

1. 白LEDが2回点滅
2. 通信準備中は消灯
3. 操作可能になると待機色を点灯

します。

待機色は、

- 通常モード：緑
- 速度調整モード：黄

です。

### Wi-Fi接続

1. ATOM Liteの電源を入れる
2. Wi-Fi一覧を開く
3. `AtomServo_XXXXXX` 形式のSSIDへ接続する
4. コード内で設定したパスワードを入力する
5. ブラウザで `http://192.168.4.1/` を開く

SSID末尾は機器ごとに自動生成されます。

- SSID末尾の `XXXXXX` は、`ESP.getEfuseMac()` から取得した値の下位24bitを6桁の16進数にしたものです
- 完全なMACアドレスはSSIDには使用していません
