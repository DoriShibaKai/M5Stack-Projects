# M5Stick-Projects

## 使用したもの

- M5StickS3・・・本体
- USB Type-Cケーブル・・・書き込み・充電用

## 使い方

### ペアリング

- AtomS3Lite本体ボタンを約4秒長押しすると、青色に点滅し約15秒間ペアリング待機になります。
- M5StickS3でKEY1を約2秒長押しします。
- KEY1を押したままKEY2を短押しすると接続先一覧が表示されます。
- 登録したい機器を選択し、KEY2を短押しすると登録されます。
- 一度登録すると、以後は自動で接続されます。

### リレー操作

- KEY1を押すとリレーを操作します。
- ダイレクト・ラッチ・タイマーモードに対応しています。

### 接続先切替

- KEY1を約2秒長押しします。
- KEY1を押したままKEY2を短押しすると接続先一覧が表示されます。
- 接続したい受信機を選択します。

## 操作方法

### ダイレクトモード

- KEY1を押している間だけリレーONになります。

### ラッチモード

- KEY1を押すたびにリレーのON／OFFが切り替わります。

### タイマーモード

- KEY1を押すと設定時間だけリレーONになります。

### タイマー時間変更

- KEY2を長押しします。
- KEY2で時間を変更します。
- KEY1で決定します。

設定した時間は受信機へ送信されます。

### 動作モード変更

- KEY2を短押しすると、
  - ダイレクト
  - ラッチ
  - タイマー
  の順に切り替わります。

## 画面表示

M5StickS3の画面には以下を表示します。

- 接続中の受信機名
- 現在の動作モード
- タイマー時間
- バッテリー残量
- ペアリング状態
- 接続状態
