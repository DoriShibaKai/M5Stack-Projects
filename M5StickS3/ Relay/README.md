# M5Stick-Projects

## 使用したもの

- M5StickS3・・・本体
- USB Type-Cケーブル・・・書き込み・充電用

## 外部スイッチ入力

- G5またはG7とGNDの間に外部スイッチを接続できます。
- 外部スイッチは本体KEY1と全く同じ動作をします。（以後、KEY1と記載している操作は外部スイッチでも同様に行えます。）
- G5、G7のどちらを使用しても動作は同じです。

## 使い方

### ペアリング

- AtomS3Lite本体ボタンを約4秒長押しすると、青色に点滅し約15秒間ペアリング待機になります。
- M5StickS3でKEY1を約2秒長押しし、そのままKEY2を短押しすると接続先選択画面が表示されます。
- 接続先選択画面でKEY1を長押しすると、新しくペアリング待機中の受信機を検索します。
- 登録したい受信機を選択し、KEY2を短押しすると登録されます。
- 一度登録すると、以後は自動で接続されます。

### 接続先切替

- KEY1を約2秒長押しし、そのままKEY2を短押しすると接続先選択画面が表示されます。
- 接続したい受信機を選択します。



## 操作方法

### リレー操作

- KEY1を押すとリレーを操作します。
- ダイレクト・ラッチ・タイマーモードに対応しています。

### ダイレクトモード

- KEY1を押している間だけリレーONになります。

### ラッチモード

- KEY1を押すたびにリレーのON／OFFが切り替わります。

### タイマーモード

- KEY1を押すと設定時間だけリレーONになります。

### タイマー時間変更

- KEY2を長押しします。
- KEY1またはKEY2で時間を変更します。
- 約3秒間何も操作しなければ通常画面へ戻ります。

設定した時間は受信機へ送信されます。

### 動作モード変更

- KEY2を短押しすると、
  - ダイレクト
  - ラッチ
  - タイマー
  の順に切り替わります。

## 画面表示

M5StickS3の画面には以下を表示します。

- 接続中の受信機名
- 現在の動作モード
- タイマー時間
- バッテリー残量
- ペアリング状態
- 接続状態
